## 🚀 الإعداد السريع

### 1. Supabase
1. أنشئ مشروعاً على [supabase.com](https://supabase.com)
2. اذهب إلى **SQL Editor** وشغّل ملف `supabase-schema.sql` كاملاً
3. انسخ **URL** و **anon key** من Project Settings → API

### 2. متغيرات البيئة
```bash
# أنشئ ملف .env في مجلد المشروع
VITE_SUPABASE_URL=https://xxxxxxxx.supabase.co
VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

### 3. تشغيل المشروع
```bash
npm install
npm run dev
```

### 4. حساب المدير
بعد إنشاء أول حساب، افتح Supabase SQL Editor وشغّل:
```sql
UPDATE companies SET is_admin = true WHERE email = 'admin@yourdomain.com';
```
ثم ادخل من `/admin`
