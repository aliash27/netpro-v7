## ⚠️ نقاط تحتاج انتباهاً

### 🔸 متوسطة الأهمية

**1. حذف الدفعات غير موجود**
- المستخدم لا يستطيع حذف دفعة مسجلة خطأ
- الحل المقترح: زر حذف مع تأكيد (للمالك فقط)

**2. تعدد الدفعات بنفس الشهر — حل UI فقط**
- منع التكرار موجود في الـ UI وفي schema
- لكن إذا تعطّل RLS في Supabase قد يتجاوزه
- الحل: `ON CONFLICT DO NOTHING` في insert

**3. لا يوجد إشعار "push" حقيقي**
- الإشعارات الحالية تظهر فقط عند فتح التطبيق
- لا يوجد إشعار للموبايل خارج التطبيق
- الحل المقترح: Supabase Edge Functions + cron job

**4. تاريخ انتهاء اشتراك المشترك الفردي**
- يوجد `start_date` لكن لا يوجد `end_date` صريح
- الحساب يفترض اشتراكاً مفتوحاً حتى آخر دفعة
- بعض ISPs يرغبون بتاريخ انتهاء محدد لكل مشترك

**5. Google Sheets — يعتمد على الاتصال**
- إذا غيّر المستخدم رابط Web App يتوقف
- لا يوجد retry عند فشل المزامنة
- لا يوجد تسجيل لوقت آخر مزامنة ناجحة في الواجهة
